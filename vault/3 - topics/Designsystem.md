#### Wie entwickle ich ein Designsystem?


wie baue ich ein designsystem?
Designsystem als Parameter für ai beim erstellen von Inhalten.
vielleicht anfangen mit einem simplen system für das erstellen von präsentationen?!
Welche Parameter gibt es?
	Farben
	Radien
	Fonts (und alles was damit zusammenhängt wie Textgrößen usw.)
	Linienstärke
	...
Wie eng will/muss ich meine Parameter setzen?
	Beispiel: Eine Reihe von möglichen Hintergrundbildern/ Shadern aus denen der agent auswählen kann vs Hintergründe werden jedes mal neu generiert - der agent nimmt sich die Hintergrundbilder/ Shader usw nur als ästhetische/funktionale Guideline
